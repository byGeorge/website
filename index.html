Favorite Development Language?

November 3, 2016

I was asked this when applying for a really awesome company, and I had no idea how to answer. It's not that I'm fickle, but my favorite language is always the one I'm using at the time. My first language was Apple Basic, and I loved writing programs, even though I couldn't save them. I've used a lot of languages over the years, and I've loved every one of them.

It's like choosing a favorite child. I even loved programming in assembly. C was harder to learn, but extremely useful. Java isn't very good at anything in particular, but it does do almost everything and is still occasionally useful. I just learned JavaScript, and I love all of the cool things it can do. I'm teaching my 11-year-old to code in Ruby because it's THAT simple to learn. The asteroids game that's going to hang out on the 80's mode is going to be written in Python. Why? Because Python is awesome and Pygame is making game creation easy.
